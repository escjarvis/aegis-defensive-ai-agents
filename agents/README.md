# Agents

This directory contains all agent implementations organized by cognitive level.

- `system1/`: On-box, fast reactive agents (e.g. Inline Protective Agent, Enforcer)
- `system2/`: Off-box reasoning and intelligence agents
- `system3/`: Predictive analysis and swarm-based agents
- `meta/`: Agent Builder, Governance, and coordination agents
- `research/`: AutoResearch and threat intelligence agents
